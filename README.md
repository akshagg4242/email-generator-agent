# Email Generator Agent

Status: Completed and verified.

## Overview

Email Generator Agent is an n8n-based automation workflow that generates professional emails from user-provided keywords and input.

The workflow converts short prompts into formatted email content.

---

## Features

- Keyword-based email generation
- Automated email formatting
- Prompt processing
- Workflow automation
