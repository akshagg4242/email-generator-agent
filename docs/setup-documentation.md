# Email Generator Agent Setup Guide

## Prerequisites

- n8n installed
- Required workflow credentials configured

---

## Step 1 — Import Workflow

Open n8n.

Select:

Import Workflow

Choose:

workflow/email-generator-agent-workflow.json

---

## Step 2 — Configure Workflow

Verify workflow nodes.

Configure:

- Trigger node
- Prompt input node
- Email generation node
- Output node

Save changes.

---

## Step 3 — Activate Workflow

Enable workflow.

Confirm activation.

---

## Step 4 — Test Email Generation

Example Input:

Write a professional email requesting project deadline extension.

Expected Output:

Generated formatted email.

---

## Step 5 — Verify Execution

Open:

Executions

Confirm successful workflow execution.

---

## Troubleshooting

No output:
- Verify node configuration

Generation issue:
- Check workflow execution logs

Credential issue:
- Reconfigure credentials

---

## Status

Workflow imported and verified successfully.
